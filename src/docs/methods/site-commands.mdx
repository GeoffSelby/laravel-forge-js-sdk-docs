## Execute a command

### Method

`forge.commands.execute(server_id, site_id, payload)`

### Usage

```js lineNumbers=true
const forge = new Forge('API_TOKEN');

await forge.commands.execute(server_id, site_id, payload);
```

### Payload

```js lineNumbers=true
{
  command: 'ls -la';
}
```

#### Payload Parameters

| Key     | Description        |
| ------- | ------------------ |
| command | Command to be run. |

### Usage
