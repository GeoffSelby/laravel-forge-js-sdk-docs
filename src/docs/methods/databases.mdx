## Create a database

### Method

`forge.database.create(server_id, payload)`

### Usage

```js lineNumbers=true
const forge = new Forge('API_TOKEN');

const database = await forge.database.create(server_id, payload);
```

### Payload

```js lineNumbers=true
{
  name: 'forge',
  user: 'forge',
  password: 'dolores'
}
```

#### Payload Parameters

| Key      | Description                                                                                                                 |
| -------- | --------------------------------------------------------------------------------------------------------------------------- |
| name     | The name of the database.                                                                                                   |
| user     | This field is optional. If passed, it will be used to create a new Database User with access to the newly created database. |
| password | This field is only required when a `user` value is given.                                                                     |

### Example Response

```json lineNumbers=true
{
  "database": {
    "id": 1,
    "name": "forge",
    "status": "installing",
    "created_at": "2016-12-16 16:12:22"
  }
}
```
