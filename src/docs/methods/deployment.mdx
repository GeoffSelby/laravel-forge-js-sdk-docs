## Enable quick deployment

### Method

`forge.deployment.enable(server_id, site_id)`

### Usage

```js lineNumbers=true
const forge = new Forge('API_TOKEN');

await forge.deployment.enable(server_id, site_id);
```

---

## Get deployment script

### Method

`forge.deployment.getScript(server_id, site_id)`

### Usage

```js lineNumbers=true
const forge = new Forge('API_TOKEN');

const script = await forge.deployment.getScript(server_id, site_id);
```

> The response is a string for this request.

---

## Update deployment script

### Method

`forge.deployment.updateScript(server_id, site_id, payload)`

### Usage

```js lineNumbers=true
const forge = new Forge('API_TOKEN');

await forge.deployment.updateScript(server_id, site_id, payload);
```

### Payload

```js lineNumbers=true

  content: 'CONTENT_OF_THE_SCRIPT';
}
```
