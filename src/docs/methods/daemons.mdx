## Create a daemon

### Method

`forge.daemons.create(server_id, payload)`

### Usage

```js lineNumbers=true
const forge = new Forge('API_TOKEN');

const daemon = await forge.daemons.create(server_id, payload);
```

### Payload

```js lineNumbers=true
{
  command: 'COMMAND',
  user: 'root',
  directory: '/home/forge/foo.com'
}
```

#### Payload Parameters

| Key       | Description                                  |
| --------- | -------------------------------------------- |
| command   | The command to run when starting the daemon. |
| user      | The server user to run the command as.       |
| directory | The directory to run the command in.         |

### Example Response

```json lineNumbers=true
{
  "daemon": {
    "id": 1,
    "command": "COMMAND",
    "user": "root",
    "directory": "/home/forge/foo.com",
    "status": "installing",
    "created_at": "2016-12-16 15:46:22"
  }
}
```
