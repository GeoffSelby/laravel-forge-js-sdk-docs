## Start a service

### Method

`forge.services.start(server_id, payload)`

### Usage

```js lineNumbers=true
const forge = new Forge('API_TOKEN');

await forge.services.start(server_id, payload);
```

### Payload

```js lineNumbers=true
{
  service: 'service-name';
}
```

#### Payload Parameters

| Key     | Description                       |
| ------- | --------------------------------- |
| service | The name of the service to start. |

## Install Blackfire

### Method

`forge.services.installBlackfire(server_id, payload)`

### Usage

```js lineNumbers=true
const forge = new Forge('API_TOKEN');

await forge.services.installBlackfire(server_id, payload);
```

### Payload

```js lineNumbers=true
{
  server_id: '...',
  server_token: '...'
}
```

#### Payload Parameters

| Key          | Description                                                              |
| ------------ | ------------------------------------------------------------------------ |
| server_id    | Your [Blackfire Server ID](https://blackfire.io/account/credentials)     |
| server_token | Your [Blackfire Server Token] (https://blackfire.io/account/credentials) |
