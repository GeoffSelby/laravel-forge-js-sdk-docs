## List all regions

### Method

`forge.regions.list()`

### Usage

```js lineNumbers=true
const forge = new Forge('API_TOKEN');

const regions = await forge.regions.list();
```

### Example Response

```json lineNumbers=true
{

    "ocean2": [

        "id": "ams2",
```
