## List deployments

### Method

`forge.deploymentHistory.list(server_id, site_id)`

### Usage

```js lineNumbers=true
const forge = new Forge('API_TOKEN');

const deploymentHistory = await forge.deploymentHistory.list(
  server_id,
  site_id,
);
```

```json lineNumbers=true
      "id": 71,
```

## Get a deployment

### Method

`forge.deploymentHistory.get(server_id, site_id, deployment_id)`

### Usage

```js lineNumbers=true
const forge = new Forge('API_TOKEN');

const deployment = await forge.deploymentHistory.get(
  server_id,
  site_id,
  deployment_id,
);
```

### Example Response

```json lineNumbers=true
{
  "deployment": {
    "id": 71,
    "server_id": 196,
    "site_id": 110,
    "type": 4,
    "commit_hash": "1aa50f0e4c49fed3a2335e866b03d4178ab93c4e",
```

## Get deployment output

### Method

`forge.deploymentHistory.output(server_id, site_id, deployment_id)`

### Usage

```js lineNumbers=true
const forge = new Forge('API_TOKEN');

const deploymentOutput = await forge.deploymentHistory.output(
  server_id,
  site_id,
  deployment_id,
);
```

### Example Response

```json lineNumbers=true
{
  "output": "Thu 05 Nov 2020 12:42:30 PM UTC\nFrom github.com:laravel/laravel\n * branch..."
}
```
