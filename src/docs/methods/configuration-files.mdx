## Get Nginx configuration

### Method

`forge.config.getNginx(server_id, site_id)`

### Usage

```js lineNumbers=true
const forge = new Forge('API_TOKEN');

const nginxConfig = await forge.config.getNginx(server_id, site_id);
```

### Example Response

The response is a string for this request.

---

## Update Nginx configuration

### Method

`forge.config.updateNginx(server_id, site_id, payload)`

### Usage

```js lineNumbers=true
const forge = new Forge('API_TOKEN');

const updatedConfig = await forge.config.updateNginx(
  server_id,
  site_id,
  payload,
);
```

### Payload

```js lineNumbers=true
{
  content: 'CONTENT';
}
```
