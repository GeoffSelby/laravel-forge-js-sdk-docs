## Get server logs

### Method

`forge.logs.get(server_id, file_type)`

### Usage

```js lineNumbers=true
const forge = new Forge('API_TOKEN');

const logs = await forge.logs.get(server_id, file_type);
```

### File Types

- `nginx_access`
- `nginx_error`
- `database`
- `php7x` (where `x` is a valid version number, e.g. `php71`) or `php56`
