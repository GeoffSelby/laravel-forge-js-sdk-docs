## Create a rule

### Method

`forge.firewallRules.create(server_id, payload)`

### Usage

```js lineNumbers=true
const forge = new Forge('API_TOKEN');

const rule = await forge.firewallRules.create(server_id, payload);
```

### Payload

```js lineNumbers=true
{
  name: 'rule name',
  ip_address: '192.168.1.1',
  port: 88,
  type: 'allow'
}
```

#### Payload Parameters

| Key        | Description                                     |
| ---------- | ----------------------------------------------- |
| name       | The name of the new firewall rule.              |
| ip_address | The ip address to either allow or deny.         |
| port       | The port that the rule should apply to.         |
| type       | You may specify allow or deny as the rule type. |

### Example Response

```json lineNumbers=true
{
  "rule": {
    "id": 4,
    "name": "rule",
    "port": 123,
    "type": "allow",
    "ip_address": null,
    "status": "installing",
    "created_at": "2016-12-16 15:50:17"
  }
}
```
