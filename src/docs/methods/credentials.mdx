## List all credentials

### Method

`forge.credentials.list()`

### Usage

```js lineNumbers=true
const forge = new Forge('API_TOKEN');

const credentials = await forge.credentials.list();
```

### Example Response

```json lineNumbers=true
{
  "credentials": [
    {
      "id": 1,
      "type": "ocean2",
      "name": "Personal"
    }
  ]
}
```
